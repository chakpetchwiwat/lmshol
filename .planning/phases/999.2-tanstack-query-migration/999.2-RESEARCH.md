# Phase 999.2: TanStack Query Migration - Research

## Current Frontend Reality

### App entry
- `elearning-webapp/src/main.jsx` currently mounts:
  - `React.StrictMode`
  - `BrowserRouter`
  - `App`
- There is no existing TanStack Query provider

### Existing hooks structure
- `src/hooks/useAccessibleOverlay.js`
- `src/hooks/useConfirm.js`

This means query/mutation hooks will be a new architectural layer rather than an incremental extension of an existing pattern.

## Do Not Treat As Standard Cached Queries

- Lesson document access URLs
- Any other signed or expiring access token flow
- Pure UI state
- Transient draft form state
