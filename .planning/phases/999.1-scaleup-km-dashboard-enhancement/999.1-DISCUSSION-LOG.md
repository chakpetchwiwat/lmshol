# Discussion Log: Phase 999.1 - Scaleup KM Dashboard Enhancement

**Date:** 2026-04-17
**Participants:** Admin (User), the agent (Builder)

## Q&A Session

### Area 1: Skill Gap Visualization & Logic
**Q: How should we map courses/quizzes to the 6 core categories?**
- **Options:** 1. Tag-based, 2. Strict Hierarchy
- **Selection:** 2. Strict Hierarchy (Each course belongs to exactly one category)

**Q: How should we visualize the comparison on the Radar Chart?**
- **Options:** 1. Static Target (100%), 2. Dynamic Profile
- **Selection:** 1. Static Target (100% for all users)

**Q: Should a user's proficiency be calculated as Average or Mastery?**
- **Options:** 1. Average, 2. Mastery (Highest score)
- **Selection:** 2. Mastery (Highest / most recent highest attempt)
- **Note:** Also requested to show total pass / non pass / attend / not attend / complete in this context.

### Area 2: Department Performance Benchmarking
**Q: Which metric should determine the rank in the Department Leaderboard?**
- **Options:** 1. Completion Rate, 2. Total Points, 3. Engagement Index
- **Selection:** 1. Completion Rate (%)

**Q: Should researchers and planners build the leaderboard to show Lifetime or Periodic stats?**
- **Options:** 1. Lifetime, 2. Periodic (Selectable)
- **Selection:** 2. Periodic (Selectable range like Month/Quarter)

### Area 3: Risk Management & ROI
**Q: What should trigger a user appearing on the "Risk Identification" list?**
- **Options:** 1. Deadline-driven, 2. Inactivity, 3. Performance-fail
- **Selection:** 1. Deadline-driven (Mandatory courses overdue or approaching deadline)

**Q: How should we visualize Incentive ROI (Points vs Output)?**
- **Options:** 1. Dual-Axis Trend, 2. Redemption Table
- **Selection:** Both 1 & 2 (Trend Chart and Redemption Table)

## Summary of Decisions
The dashboard will prioritize **Performance Transparency** via completion rates and **Strategic Competency** via mastery-based radar charts. The ROI layer will directly link the Incentive module to Learning outcomes using dual-axis visualizations.
